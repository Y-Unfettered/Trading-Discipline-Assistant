# 交易纪律助手

本地运行的持仓、成交记录、机械纪律检查与 Codex 复盘工具。

## 启动

```powershell
npm start
```

浏览器打开：

```text
http://127.0.0.1:3768
```

## Codex 分析

应用不需要 OpenAI API Key。它通过本机安装的 Codex CLI 发起只读分析。

首次使用前登录：

```powershell
codex login
```

如果暂时不登录，也可以点击“仅生成复盘包”，然后让 Codex 桌面版读取：

```text
reports/review-latest.json
```

## 数据位置

- `data/store.json`：账户、持仓和成交记录
- `reports/`：复盘数据包和分析报告

这两个目录以及日志、备份、券商截图和一次性导入脚本均被 Git 忽略，只保存在本机。请勿使用 `git add -f` 强制提交个人交易数据。

程序仅监听 `127.0.0.1`，不会主动开放给局域网。

## 行情说明

行情刷新使用第三方公开接口，可能延迟、限流或发生字段变化。它只用于记录和复盘，不应替代券商行情作为下单依据。
